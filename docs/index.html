School Communication Feedback

A parent-focused space to discuss school communications constructively and improve clarity, consistency, and engagement.

Home News Policies
Purpose

This site exists to help parents discuss school communications in a way that is practical and solution-focused. The aim is to identify where messages are unclear, inconsistent, or hard to find — and suggest improvements.

Guidelines
  • Be respectful. Critique the message, not individuals.
  • Keep it specific: quote the part that’s confusing and explain why.
  • Suggest an improvement (wording, structure, timing, where it should live).
  • Avoid personal information about students, staff, or families.
How comments work

Comments are powered by GitHub Discussions (via giscus). To comment, you’ll need a GitHub account. If you prefer not to comment publicly, you can still read along.
